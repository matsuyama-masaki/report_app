# Report App

FastAPI と Next.js を使用した、シンプルかつ実用的なレポート管理システムです。
インフラからフロントエンドまで、モダンな技術スタックを用いたフルスタック開発の構成例として構築しました。

## 🛠 技術スタック

### Backend
- **Framework**: FastAPI (Python 3.12)
- **ORM**: SQLModel (SQLAlchemyベース)
- **Database**: PostgreSQL (Docker)
- **Documentation**: Swagger UI (OpenAPI)

### Frontend
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS

### Infrastructure
- **Container**: Docker / Docker Compose
- **Tool**: Cursor Editor (AI-driven development)
